# Taygun Bulmus

## Running Locally

```bash
# Install Mise: https://mise.jdx.dev/installing-mise.html#apt
# Update Mise:
# mise self-update

### Install dependencies
mise install
# Run hugo server
mise run server
### Updating theme
# To update theme to latest release, run:
mise run update
# To update theme to latest commit from `main` brnach, run:
mise run update-to-main
```
